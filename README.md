# DaJet Metadata

Библиотека для чтения и анализа метаданных из баз данных 1С:Предприятие 8.

Поддерживаются Microsoft SQL Server и PostgreSQL (специальная сборка для 1С).

Поддерживаются следующие объекты метаданных 1С:Предприятие 8:
- Свойства основной конфигурации и расширений
- Общие реквизиты
- Определяемые типы
- Планы обмена
- Константы
- Перечисления
- Справочники
- Документы
- Планы видов характеристик
- Регистры сведений
- Регистры накопления
- Планы счетов
- Регистры бухгалтерии
- Задачи
- Бизнес-процессы
- Таблицы регистрации изменений

Расширения основной конфигурации поддерживаются, начиная с версии 1С:Предприятие 8.3.12 и выше.
